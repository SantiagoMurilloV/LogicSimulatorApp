# LogiFlow — Circuit Logic Simulator

Interactive digital logic circuit simulator — build, simulate, and analyze boolean logic circuits in your browser.

## Features

- Drag-and-drop circuit building with AND, OR, NOT, NAND, NOR, XOR gates
- Real-time simulation with signal propagation
- Boolean equation input/output in canonical form (SOP notation)
- Truth table generation and reverse engineering
- PWA support — installable and works offline
- Mobile-friendly responsive design

## Tech Stack

- Next.js 16 + React 19
- React Flow for circuit visualization
- Zustand for state management
- Tailwind CSS 4
- TypeScript

## Getting Started

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
